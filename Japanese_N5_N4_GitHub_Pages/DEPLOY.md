# วิธีอัปโหลดแบบเร็ว

1. แตกไฟล์ ZIP
2. เปิด GitHub และสร้าง Repository ใหม่
3. กด `Add file` → `Upload files`
4. ลาก `index.html`, `.nojekyll` และ `README.md` ลงไป
5. กด `Commit changes`
6. ไปที่ `Settings` → `Pages`
7. เลือก `Deploy from a branch`
8. เลือก `main` และ `/ (root)` แล้วกด `Save`

ลิงก์จะมีรูปแบบ:

`https://USERNAME.github.io/REPOSITORY/`
