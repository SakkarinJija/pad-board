# Japanese N5–N4 Study Dashboard

เว็บแดชบอร์ดติดตามการเรียนภาษาญี่ปุ่น N5–N4 ระยะเวลา 8 เดือน

## เปิดใช้งานด้วย GitHub Pages

1. สร้าง GitHub Repository ใหม่
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่หน้าแรกของ Repository
3. เปิด `Settings`
4. เลือก `Pages`
5. ที่ `Build and deployment` เลือก `Deploy from a branch`
6. เลือก Branch `main` และ Folder `/ (root)`
7. กด `Save`
8. รอสักครู่ แล้วเปิดลิงก์ GitHub Pages ที่ระบบแสดง

## การบันทึกข้อมูล

ข้อมูลจะบันทึกไว้ใน Local Storage ของเบราว์เซอร์

- ใช้งานได้ทันทีโดยไม่ต้องมีฐานข้อมูล
- ควรใช้อุปกรณ์และเบราว์เซอร์เดิม
- ใช้ปุ่ม `สำรองข้อมูล` ในหน้าเว็บเพื่อดาวน์โหลดไฟล์ JSON เก็บไว้

## ไฟล์หลัก

- `index.html` — หน้าเว็บไซต์และระบบแดชบอร์ดทั้งหมด
- `.nojekyll` — ป้องกัน GitHub Pages ประมวลผลผ่าน Jekyll
